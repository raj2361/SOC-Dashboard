# 🛡️ SOC Dashboard

A Security Operations Center (SOC) Dashboard built using **Python**, **Flask**, **SQLite**, **HTML**, **CSS**, and **Chart.js**.

## 📌 Overview

This project simulates a real SOC analyst dashboard where security alerts can be monitored, investigated, and managed.

The dashboard allows analysts to:

- View security alerts
- Filter alerts by severity and status
- Search alerts
- Investigate alert details
- Update investigation status
- Visualize alert statistics
- Store alerts in a SQLite database
